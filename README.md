# 🎵 From The Heart

A full-stack music streaming platform built for independent artists.

## 🚀 Features

- Artist pages with dynamic routing
- Music streaming with play tracking
- Album and track organization
- Search functionality
- Authentication (JWT)
- Admin + Artist system (upload music, manage content)
- Cloudinary media storage

## 🧠 Tech Stack

### Frontend
- Next.js (Pages Router)
- Material UI
- Axios

### Backend
- Node.js
- Express
- MongoDB (Mongoose)
- Cloudinary (media storage)
- Stripe (payments - optional)

## 🔐 Roles

- **User** → listens to music
- **Artist** → uploads music & manages profile
- **Admin** → creates artist accounts
